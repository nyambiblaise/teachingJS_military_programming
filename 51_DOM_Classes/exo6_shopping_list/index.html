Welcome to our app

We've got a shopping list. You can highlight items on the shopping list by clicking on an item.

When you click on an item, we automatically call the handleItemClick function with the item (the li) that was clicked.

Complete the handleItemClick function such that it adds the highlighted class when the li is not already highlighted and removes it when it's already highlighted.

We've made the first item highlighted by default so that you can visualize it.
