This page contains 3 tabs. Every tab has a class tab. The one that is currently activated has the class active. When you click on any of the tabs, we call the following functions in order:

1- deactivateAllTabs(): this function receives no parameters and is supposed to remove the class active from every .tab element (or, since only one tab can be active in this challenge, you can remove the class active from the currently active tab).

2- activateClickedTab(tab): this function receives the tab element that was clicked by the user and is supposed to add the class active to this element.
